# SITIO WEB CON ANGULAR.JS NODE.JS

	- Sitio web creado para crear, leer, eliminar y editar contenido en  
	base de datos MongoDB con ExpressJs AngularJs NodeJs.  	 

# Herramientas:
	
	- Node Js
	- Angular Js - 2 / 4 -
	- MongoDB
	- ExpressJs
	- Bootstrap v3.1.5
	- jQuery v3.2.1
	- Font Awesome 4.3.0
	- html5 css3
